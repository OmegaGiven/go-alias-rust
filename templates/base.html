Aliases

Use Aliases to create short browser routes. Type an alias into the URL bar, like /gh, to open the saved destination.

The Add Shortcut button on the Aliases page can save local browser aliases or global server aliases.

SQL Manager

Use SQL to save database connections, open one or more connection tabs, browse tables, inspect functions, run queries, and keep output history per connection tab.

Click a table to open a live paginated table browser. Use filters to narrow rows and save common table filters.

Click a function in the Functions tab to load its definition into the editor for inspection.

Use Cmd+Enter or Ctrl+Enter to run the current SQL query.

SQL Variables

Use variables when a query needs values you may change each run.

Type a placeholder using double braces, like {{customer_id}}. The variable bar creates an input automatically.

Use + Var to add a manual variable input, then use that same name in the query.

SELECT *
FROM orders
WHERE customer_id = {{customer_id}}
  AND status = '{{status}}';

Requests

Use Requests to build and send HTTP requests, save requests into folders, import Postman collections, manage variable sets, and inspect response history.

Use the Curl tab to paste a curl command and populate the request builder. Use View Curl to see the equivalent command for the current request.

Open in Inspector sends the latest response body to the Inspector page for deeper JSON analysis.

Use Cmd+Enter or Ctrl+Enter to send the current request.

Inspector

Use Inspector to paste, load, prettify, validate, and inspect JSON or text payloads.

Search and select nodes to understand nested objects, arrays, paths, and large response structures.

Floating Tools

The hamburger menu opens Appearance, Calculator, JWT Decoder, and Scratch Pad.

Floating windows can stay open while you work. Drag their header to move them, or double-click an outer edge to center supported windows.

Appearance

Use Appearance to switch light, dark, or custom themes and adjust font sizes, font family, accent color, nav height, and global element margin.

JWT Decoder

Paste a JWT or API token to decode header and payload values. Date-like claims are shown in a readable format when possible.

Scratch Pad

Use Scratch Pad for temporary notes, queries, and snippets. Entries are saved automatically and can be switched with compact tabs.
